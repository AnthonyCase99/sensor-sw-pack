#define CN0397_OPTIMAL_CONC_BLUE_CFG   (26909)

Optimal level of light for the blue photodiode.

The default value is taken from the PAR(Photosynthetically Active Radiation) curve which plants react to.

§ CN0397_SENSOR_ID_CFG

#define CN0397_SENSOR_ID_CFG   (0x01u)

ID of sensor.

Unique identifier for this particular sensor. This is used in the Bluetooth data packet to alert the Android application of a particular instance of a sensor. This value is a 7-bit number.
