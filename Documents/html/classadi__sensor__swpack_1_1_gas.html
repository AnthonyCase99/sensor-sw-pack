virtual SENSOR_RESULT adi_sensor_swpack::Gas::start ( )

Place the gas sensor into continuous measurement mode.

Puts the gas sensor in continuous measurement mode. If this function is not called, the gas sensor will only make a meaurement when getPPM is called. This will take more time, but save power.
