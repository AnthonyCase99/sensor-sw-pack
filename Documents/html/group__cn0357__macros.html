§ CN0357_FEEDBACK_RESISTOR

#define CN0357_FEEDBACK_RESISTOR   (9230.76)

Feedback resistor.

This is the value that the rheostat needs to be configured to for proper operation (ohms). This value is dervied from equation 2 (located on page 2) on the CN0357 datasheet. Which is:

Rf = Vref / (Range * Sensitivty)

Where the sensitivty is given above and the range is 2000 PPM.

§ CN0357_SENSOR_SENSITIVTY

#define CN0357_SENSOR_SENSITIVTY   (0.0000000065)

Sensor sensitivity.

This value is taken from Table 1 in the CN0357 datasheet. The units are amperes per PPM. It is used when computing the gas concentration in PPM in the getPPM function.
