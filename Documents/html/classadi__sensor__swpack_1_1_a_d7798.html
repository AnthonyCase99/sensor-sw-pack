§ init()

virtual SENSOR_RESULT adi_sensor_swpack::AD7798::init ( )
virtual

Initializes the ADC.

Returns
SENSOR_RESULT. This value can be set using SET_SENSOR_ERROR(type, error).

Init function initializes the ADC and underlying peripheral. This API has to be implemented by all ADC classes. A read of the ADC ID value is performed to confirm the ADC is responding correctly.

Note
An assumption is made that the SPI bus specified in the configuration file will not be modified by another sensor. As a result, it will only be configured during this function and not on a transaction by transaction basis.

Implements adi_sensor_swpack::ADC.

virtual SENSOR_RESULT adi_sensor_swpack::AD7798::reset ( )
virtual

Reset the ADC.

Returns
SENSOR_RESULT. This value can be set using SET_SENSOR_ERROR(type, error).

This function resets all ADC registers to their default values. The user must wait 500 us after a reset before accessing an on chip register. This API has to be implemented by all ADC classes.
